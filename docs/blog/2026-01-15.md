### LLM Architectures
- **[DInf-Grid: A Neural Differential Equation Solver with Differentiable Feature Grids](https://arxiv.org/pdf/2601.10715v1)**  
  DInf-Grid introduces an innovative method for solving differential equations efficiently with a neural network architecture that emphasizes grid-based representations and multi-resolution decompositions.

- **[MatchTIR: Fine-Grained Supervision for Tool-Integrated Reasoning via Bipartite Matching](https://arxiv.org/pdf/2601.10712v1)**  
  This framework enhances large language models' interactions with tools through fine-grained rewards, significantly improving reasoning precision over long sequences.

### Computer Vision
- **[Toward Ultra-Long-Horizon Agentic Science: Cognitive Accumulation for Machine Learning Engineering](https://arxiv.org/pdf/2601.10402v1)**  
  This study introduces a framework for autonomous agents that enhances long-term strategic coherence in engineering tasks, using cognitive accumulation techniques.

- **[SatMap: Revisiting Satellite Maps as Prior for Online HD Map Construction](https://arxiv.org/pdf/2601.10512v1)**  
  A new method for online HD map estimation that leverages satellite imagery with camera observations, improving accuracy and robustness.

### Robotics and Control
- **[FlowAct-R1: Towards Interactive Humanoid Video Generation](https://arxiv.org/pdf/2601.10103v1)**  
  Introduces a real-time system for generating humanoid videos that combines low-latency responsiveness with fine-grained control over characters' behaviors.

- **[CoF-T2I: Video Models as Pure Visual Reasoners for Text-to-Image Generation](https://arxiv.org/pdf/2601.10061v1)**  
  A novel model that seamlessly integrates video reasoning into text-to-image generation, improving performance over traditional models.

### Safety and Robustness
- **[ReasAlign: Reasoning Enhanced Safety Alignment against Prompt Injection Attack](https://arxiv.org/pdf/2601.10173v1)**  
  This work presents a model-level solution designed to improve safety alignment against indirect prompt injection attacks through structured reasoning steps.

- **[How does downsampling affect needle electromyography signals? A generalisable workflow for understanding downsampling effects on high-frequency time series](https://arxiv.org/pdf/2601.10191v1)**  
  Explores the impact of downsampling methods on the signal quality in medical data, creating a robust analysis framework for feature extraction.
